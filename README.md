# Container Depot App

Custom Frappe app for Container and ISO Tank Depot Management System.

## Installation

```bash
cd /home/frappe/frappe-bench
bench get-app container_depot <path_to_container_depot>
bench install-app container_depot
```

## Features

- Container asset tracking
- Voucher/Gate management (Bon Bongkar QR system)
- Inspection system with photo evidence
- Maintenance & Repair workflow
- Cleaning queue management
- Split-billing support
